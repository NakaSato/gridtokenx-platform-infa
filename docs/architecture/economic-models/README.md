## 💰 Tri-Token System

### GRX (Governance & Staking)
- **Supply:** Fixed at 1,000,000,000 GRX
- **Purpose:** Governance, staking, value capture
- **Validator Requirement:** 10,000 GRX minimum stake

### GRID (Energy Utility)
- **Supply:** Elastic (mint-and-burn)
- **Purpose:** Represents 1 kWh of certified renewable energy
- **Backing:** Proof of Provenance (PoP) certified

### Currency Token
- **Purpose:** Settlement medium for trades
- **Standard:** SPL Token-2022
- **Stability:** Pegged to local currency units
